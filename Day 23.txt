ResultSetMetadata Interface:
The metadata means data about data i.e. We can get further information from the data.
This interface will provide us table information like total number of column, column type, column size, is readable or writable,
schema name etc. 

DatabaseMetadata Interface:
This interface provides method to get metadata of a database such as database product name, database product version, driver name,
number of tables, number of views etc.

MySQL Data Types:

String & text - Fixed length(TINYTEXT,TEXT, MEDIUMTEXT,LONGTEXT)   Variable length(VAECHAR, ) 
Date & Time - YEAR, DATE, TIME, DATETIME, TIMESTAMP
Blob & Json - BLOB, MEDIUMBLOB, LONGBLOB, JSON
