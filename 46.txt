CSS(Cascading Style Sheet):
CSS is used to enhance the look and feel of the webpage. Using css we can change height, width, color, font, margin, border etc.

ways to add css:
i)Inline css
ii)Internal css
iii)External css 


Selectors:
CSS selectors are used to select the content you want to style. CSS selector selects HTML elements according to its id, class, type, attribute etc.

i)Element Selector  - p{}, div{}
ii)Id Selector - #para1
iii)Class selector - .test{}
iv)Group selector - p,div,h1{}
v)Universal selector - *{}

Box model:The css box model is essentially a box that wraps around every html element. It consists of content, padding, borders and margins. 
It is used to develop the design and structure of a web page. We can personalize the layout of the component using box model.

i)Content : The content of the element where the text and images appear.
ii)Padding : It is the space around the content.
iii)Border: A border is the space aroung the padding and content.
iv)Margin : Clears an area outside the border.
